# Frontend - Aplikasi Notes

Frontend untuk Aplikasi Notes menggunakan vanilla HTML, CSS, dan JavaScript.

## Fitur
- CRUD catatan (Tambah, Lihat, Edit, Hapus)
- Pencarian catatan
- Statistik catatan (total & hari ini)
- Dark theme modern dengan animasi
- Responsive design
- Toast notification
- Konfirmasi hapus dengan modal

## Cara Menjalankan
1. Install dependencies: `npm install`
2. Jalankan: `npm start`

## Catatan
- Pastikan backend sudah berjalan terlebih dahulu
- Sesuaikan `API_URL` di file `app.js` dengan URL backend kamu
